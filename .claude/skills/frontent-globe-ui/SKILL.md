# Frontend Globe UI

Use this skill when working on the React frontend.

Rules:

- Use React 19 + Vite + TypeScript
- Keep Tailwind styling clean and minimal
- Use react-globe.gl for globe rendering
- Keep timeline slider at the bottom
- Keep event modal readable and compact
- Prefer reusable components
- Do not add auth or user settings flows
- Do not bypass backend API with direct JSON imports in UI
